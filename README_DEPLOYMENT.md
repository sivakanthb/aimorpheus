# 🚀 AIMorpheus - AI Learning Portal

Master the Future of Intelligence. Your personalized journey to mastering artificial intelligence.

## ✨ Features

- **Personalized Learning Paths** - Get AI-powered recommendations based on your preferences
- **7 Comprehensive Modules** - From AI Fundamentals to Generative AI
- **Smart Matching Algorithm** - 4-factor weighted scoring for perfect path selection
- **Hands-On Resources** - Links to tools, frameworks, and practical learning platforms
- **Community Blog** - Share insights and discuss AI with other learners
- **Featured Articles & Books** - Stay updated with latest AI research and recommendations
- **Beautiful Dark UI** - Soothing, eye-friendly interface with modern design

## 🛠 Tech Stack

- **Framework:** Next.js 16.1.6 with App Router
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS 4
- **Runtime:** Node.js 24.14.0+
- **Package Manager:** npm 11.9.0+

## 📦 Installation

### Prerequisites
- Node.js 24.14.0 or higher
- npm 11.9.0 or higher

### Local Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/aimorpheus.git
   cd aimorpheus
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   - Navigate to http://localhost:3000
   - Start exploring your personalized AI learning journey!
